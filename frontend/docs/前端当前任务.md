学生 CSV 导入接口
接口地址：
POST http://127.0.0.1:8000/api/teacher/students/import-csv
请求方式：
Content-Type: multipart/form-data
请求字段：
字段
类型
必填
说明
file
File
是
CSV 文件
前端示例：
const formData = new FormData()
formData.append("file", file)

const res = await fetch("http://127.0.0.1:8000/api/teacher/students/import-csv", {
  method: "POST",
  body: formData
})

const data = await res.json()
CSV 模板支持英文表头：
student_no,student_name,grade,class_name,status
20300101,张三,6,1,active
20300102,李四,7,2,active
也支持中文表头：
学号,姓名,年级,班级,状态
20300101,张三,6,1,active
20300102,李四,7,2,active

成功返回：
{
  "message": "Students imported successfully",
  "count": 2,
  "created_count": 2,
  "updated_count": 0
}
前端展示建议：
•
count：本次读取到的学生总数
•
created_count：新增人数
•
updated_count：更新人数
失败返回示例：
{
  "detail": "Only .csv files are supported"
}
或：
{
  "detail": "student_no must be an 8-digit number"
}
前端需要做的校验：
•
只允许选择 .csv
•
文件不能为空
•
上传后根据 HTTP 状态码判断成功或失败
•
失败时展示 detail
注意：前端不需要解析 CSV，只需要上传文件。
